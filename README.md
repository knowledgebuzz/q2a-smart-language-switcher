A dynamic native language switcher for **Question2Answer (Q2A)** that automatically detects installed language packs and allows users to switch interface languages seamlessly.

Designed for performance, stability, and production environments.

---

## 🚀 Features

- ✅ Automatically detects installed language folders
- ✅ Smart fallback detection
- ✅ Alphabetical language sorting
- ✅ Remembers user-selected language
- ✅ Clean UI with “Select Language” placeholder
- ✅ Active language indicator with “Change Language” label
- ✅ Lightweight and optimized (no heavy reload loops)
- ✅ Works with all properly structured Q2A language packs
- ✅ Production ready & stable

---

## 📦 Installation

1. Download or clone this repository.
2. Upload the plugin folder into:

```
qa-plugin/
```

3. Go to **Admin → Plugins** and activate **Smart Language Switcher**.
4. Ensure your language folders are inside:

```
qa-lang/
```

Done ✔

## 🌐 Language Folder Naming Guide

This plugin reads installed language folders automatically.

Common folder naming examples:

| Folder Name | Displayed As |
|-------------|-------------|
| `en`        | English |
| `en-GB`     | English (UK) |
| `ar`        | Arabic |
| `fr`        | French |
| `hi`        | Hindi |
| `zh`        | Chinese |
| `ru`        | Russian |
| `es`        | Spanish |
| `de`        | German |

Some third-party language packs may use non-standard folder names (e.g., `master-fr-q2a`).  
The plugin intelligently handles most cases without renaming.

For detailed documentation and compatibility guidance:

👉 https://unitedafrica.digital/q2a-language-selector
